# E-Commerce Sales & Customer Behavior Analysis

## 📊 Project Overview
This project analyzes e-commerce sales data and customer behavior patterns using Python. The analysis includes exploratory data analysis (EDA) and data visualizations to extract business insights from a Brazilian e-commerce dataset.

## 🎯 Objectives
- Analyze order status distribution and fulfillment rates
- Understand customer distribution across different states
- Examine price distributions and product performance
- Analyze shipping costs and their impact on orders
- Identify key business metrics and trends

## 📈 Key Findings
- **Total Revenue:** $13,591,643.70
- **Total Orders:** 99,441
- **Average Order Value:** $136.68
- **Order Fulfillment Rate:** 97.6% delivered successfully
- **Top State by Customers:** São Paulo (SP)
- **Average Item Price:** $120.77
- **Average Shipping Cost:** $13.40 per item

## 📁 Project Files

### Data Files
- `olist_orders_dataset.csv` - Order information
- `olist_customers_dataset.csv` - Customer details
- `olist_order_items_dataset.csv` - Item-level order data
- `olist_products_dataset.csv` - Product information
- `olist_order_payments_dataset.csv` - Payment data
- `olist_order_reviews_dataset.csv` - Customer reviews

### Python Scripts
- `eda_analysis.py` - Exploratory Data Analysis
- `chart_states.py` - Top states visualization
- `chart_prices.py` - Price distribution analysis
- `chart_shipping.py` - Shipping cost analysis

### Visualizations
- `chart1_order_status.png` - Order status distribution
- `chart2_top_states.png` - Customer distribution by state
- `chart3_price_distribution.png` - Product price distribution
- `chart4_shipping_costs.png` - Shipping cost breakdown

## 🛠 Technologies Used

- **Matplotlib** - Data visualization

## 📊 Analysis Highlights

### Order Status Distribution
- **Delivered:** 97,598 orders (97.6%)
- **Shipped:** 2,859 orders (2.9%)
- **Cancelled:** 625 orders (0.6%)
- **Other statuses:** <0.1%

### Geographic Insights
The analysis shows concentration of customers in major metropolitan areas, with São Paulo leading in customer count and revenue generation.

### Pricing Patterns
Most products are priced between $50-$200, with a long tail of premium products extending up to $6,000+.

## 📌 How to Run

1. Install required libraries:
```bash
pip install pandas numpy matplotlib seaborn
```

2. Ensure all CSV files are in the same directory as the Python scripts

3. Run the analysis:
```bash
python eda_analysis.py
```

4. Generate visualizations:
```bash
python chart_states.py
python chart_prices.py
python chart_shipping.py
```
